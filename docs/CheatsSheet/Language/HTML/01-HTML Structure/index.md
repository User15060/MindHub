# Structure de base d’un fichier HTML

```html title="index.html" linenums="1" hl_lines="2 5"
<!-- DOCTYPE : Indique que le document est en HTML5 -->
<!DOCTYPE html>
<!-- <html> : Racine du document HTML -->
<!-- lang="fr" : Définit la langue principale du contenu -->
<html lang="fr">
  <head>
    <!-- Encodage des caractères en UTF-8 pour supporter les caractères spéciaux -->
    <meta charset="UTF-8">

    <!-- Assure la compatibilité avec Internet Explorer -->
    <meta http-equiv="X-UA-Compatible" content="IE=edge">

    <!-- Permet une mise en page responsive sur tous les appareils -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!-- Titre affiché dans l’onglet du navigateur -->
    <title>Structure en HTML</title>
  </head>

  <body>
    <!-- Corps de la page : Contenu visible par les utilisateurs -->
  </body>
</html>

<!-- Rem: Pour écrire cette structure plus vite, vous pouvez écrire "! + Enter" -->
```
